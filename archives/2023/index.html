WiFi Sensing: Behavior-Based WiFi User Dataset
The Behavior-based WiFi User Dataset for user authentication. This dataset contains the physiological characteristics captured by WiFi from 10 participants for 10 different activities. Each participant performs 20 rounds for each activity. The experiments are conducted in two different environments, the campus office, and the home apartment. The dataset can be used by fellow researchers to reproduce the original work or to further explore other machine-learning problems in the domain of WiFi sense.
2023-02-12 Rutgers University
11
WiFi Sensing: WiFi-based Object Detection
The dataset of environment-independent in-baggage object identification system leveraging low-cost WiFi. The dataset contains the extracted CSI features from 14 representative in-baggage objects of 4 different materials. The experiments are conducted in 3 different office environments with different sizes.
2023-02-11 Rutgers University
08
Vibration: Vibration-based Client Authentication
The dataset provide vibration infomation extracted from wearble device. The dataset contains x,y,z axis movement positional data collected from 2 participants in ten's hours of on desk and on wrist motion activies. We develop a series of data processing methods to mitigate the impacts of noises and body movements.
2023-02-08 New York Institute of Technology
06
mmWave: mmWave-based Activity Recognition
The mmWave-based activity dataset is used for people daily activity recognition. It contains two sub-datasets. The first dataset (FA Dataset) contains 14 common daily activities. This second one (EA Dataset) contains 5 kinds of eating activities. Our dataset can be used by fellow researchers to reproduce the original work or to further explore other machine-learning problems in the domain of mmWave signals.
2023-02-06 George Mason University
